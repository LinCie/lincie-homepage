# Design Context

## Users
General public visitors viewing a personal website. The site should feel like visiting a living garden — warm, welcoming, and accessible. Visitors are on a gentle, unhurried journey through LinCie's world, not clicking through a sterile grid.

## Brand Personality
- **Warm & Comforting**: Like home-cooked food from someone who loves you — sincere, genuine, unpretentious
- **Innocent & Honest**: Not performing cute, being cute. Genuine personality over polished performance
- **Handmade**: Crafted with care, organic and hand-placed — never sterile, corporate, or manufactured
- **Playful & Soft**: First-person voice that is fun and gentle, like talking to a close friend

### Anti-References
- Sterile grids and rigid layouts — the primary aesthetic enemy
- Corporate or generic portfolio templates
- Anything that feels manufactured, pretentious, or inauthentic

## Aesthetic Direction
- **Theme**: A living garden that mirrors the real world across two dimensions — time of day and season
- **Core metaphor**: Spring — blooming, fresh, gentle life
- **Color system**: Mint-based palette that shifts dynamically:
  - **Time of day**: Light mint morning → warm cream afternoon → soft pink evening → calm blue night
  - **Seasons**: Spring mint bloom → summer warm pink → autumn amber → winter cool blue
- **Typography**: Handwritten fonts (Caveat Variable for headings, Kalam for body) — personal, imperfect, distinctly "LinCie"
- **Texture**: Soft fabric / watercolor feel — tactile and organic
- **Motion**: Gentle swaying, floating, drifting — nothing abrupt or mechanical
- **Voice**: Playful and soft, first-person

### Key Design Elements
- **Hero flower**: Gently sways, responds to hover, evolves with seasons (blooming → full → shedding → resting)
- **Garden path navigation**: Organic journey between sections — exploration, not clicking
- **Greenhouse portfolio**: Clean work presentation within the warm garden world — professional credibility without sacrificing personality
- **Secret garden**: Hidden area rewarding curious visitors with intimate content
- **Sad wilting flower 404**: Emotional, delightful error page
- **Opt-in chimes**: Soft audio triggered by flower interaction — discovery, not intrusion

## Design Principles
1. **Nature is architecture, not decoration**: The garden metaphor structures the entire site — navigation, layout, content hierarchy, and visual language all flow from the nature theme
2. **Every element earns its place**: Nothing decorative for decoration's sake. Simplicity means removing everything that adds no value
3. **Living over static**: The site should feel like a place you visit, not a page you load — time-of-day and seasonal systems make it alive
4. **Organic over sterile**: Hand-placed layouts, slight asymmetry, gentle tilts — avoid rigid grids. The imperfection IS the charm
5. **Layered intimacy**: Public home → professional portfolio → private diary — progressive depth rewards engagement
6. **Subtle details matter**: Small intentional touches (hover states, transitions, sway animations, particle effects) add delight without clutter

## Technical Notes
- Framework: Astro (static site generator) with Tailwind CSS v4
- No JavaScript framework — vanilla TypeScript only
- No component library — fully hand-crafted design system
- Color theming via CSS custom properties with `data-time` and `data-season` attributes on `<html>`
- Accessibility target: WCAG AA standard
- Light mode only — the living world system replaces the need for dark mode
